# Nutrition Logic – v1

## 1. Modes & Tabs(Ingredient and Smoothie)

### Ingredient Tab
- User can type ingredient names and analyze them.
- User can analyze multiple ingredients at a time.
- Output: nutrition info, allergen info, health benefits for each ingredient.

### Smoothie Tab
- User can select the ingredient to make full smoothie and analyze it.
- User can include multiple ingredients at once.
- Output: combined nutrition, allergen, and health-goal feedback for the whole smoothie.
